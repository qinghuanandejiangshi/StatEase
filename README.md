# StatEase · 数据统计分析工具
StatEase 是一款面向 非计算机背景学生与科研人员 的轻量级数据统计分析软件。
无需编程基础，即可完成从数据导入、清洗到统计检验、聚类分析、可视化展示的一站式流程。

## 📌 目标：让数据分析变得更简单，让每位学生都能轻松完成科研级统计分析。

## ✨ 核心特性（Current Features）

### 📁 1. 数据管理

数据导入（Excel/CSV 等）

数据导出

数据清洗（缺失值处理、字段删除等）

### 📊 2. 统计分析功能

描述性统计（均值、方差、分布特征等）

两组比较：T 检验

多组比较：单因素方差分析（ANOVA）

相关性分析（Pearson / Spearman）

线性回归分析（Simple/Multiple）

### 📈 3. 高级分析功能

主成分分析（PCA）

K-Means 聚类（持续优化中）

### 🧪 4. 科研级可视化能力

自动生成多种专业图表（箱线图、散点图、条形图、相关矩阵等）

清晰、规范、适用于科研论文的绘图风格

## 🎯 产品定位
StatEase 的设计初衷是帮助 没有编程基础的学生 快速完成统计分析，降低学习门槛。
相比传统统计工具（如 SPSS、R、Python），StatEase 更加：
上手简单：一键导入即可分析

界面清爽：所有功能在侧栏清晰呈现

针对教学场景优化

无需记忆复杂代码

适用于：毕业论文、实验课数据处理、科研作图、课程作业分析等。
